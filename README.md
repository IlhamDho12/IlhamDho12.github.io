# Redho Ilhamdi — IT Professional Portfolio

Personal portfolio website designed to showcase my professional background, skills, certifications, and featured projects.

## 🚀 Features
*   **Dynamic Typing Animation**: Auto-typed roles in the hero section.
*   **Cursor Glow Effect**: Interactive glow element following the mouse cursor.
*   **Interactive Particle Canvas**: Lightweight custom HTML5 canvas particles rendering in the background.
*   **Animated Timeline**: Clean professional experience timeline.
*   **Verified Certifications**: Detailed layout showcasing verified certifications from dicoding, Cisco, IBM, Oracle, and Hacktiv8.
*   **Tailwind CSS Integration**: Fully responsive design leveraging Tailwind CSS utility classes.
*   **Scroll Animations**: Smooth entrance scroll-triggered animations.

## 🛠️ Tech Stack
*   **Frontend**: HTML5, Vanilla CSS3 (custom styles & animations), Vanilla JavaScript.
*   **Styling Framework**: Tailwind CSS.
*   **Icons**: FontAwesome.
*   **Fonts**: Inter & JetBrains Mono (Google Fonts).

## 💻 Local Development

1.  Make sure you have **Node.js** installed.
2.  Open your terminal in this directory.
3.  Start the local server:
    ```bash
    npm run dev
    ```
4.  Open your browser and navigate to **[http://localhost:3002](http://localhost:3002)**.
